<task type="auto">
  <name>Task 3: Create temporal alignment module for speaker-word matching</name>
  <files>src/stages/alignment.py</files>
  <action>
Create alignment module that matches Whisper word timestamps with pyannote speaker segments using temporal overlap.

Implementation requirements:

1. Import from existing modules:
   - from src.stages.transcription import TranscriptionResult, WordInfo, SegmentInfo
   - from src.stages.diarization import DiarizationResult, SpeakerTurn
   - from src.config.settings import ASR_CONFIDENCE_THRESHOLD

2. Create dataclasses:
   - AlignedWord: word (str), start (float), end (float), speaker (str), confidence (float), needs_review (bool)
   - AlignedSegment: id (int), text (str), start (float), end (float), speaker (str), confidence (float), needs_review (bool), words (List[AlignedWord])

3. Create helper function `find_speaker_for_word(word_start: float, word_end: float, turns: List[SpeakerTurn]) -> str`:
   - Calculate overlap duration with each speaker turn: max(0, min(word_end, turn.end) - max(word_start, turn.start))
   - Return speaker with maximum overlap
   - If no overlap found (max_overlap == 0), find nearest speaker:
     - Calculate distance to each turn: min(abs(word_start - turn.end), abs(word_end - turn.start))
     - Return speaker with minimum distance
   - If still no speaker found, return "UNKNOWN"

4. Create `align_transcript_with_speakers(transcription: TranscriptionResult, diarization: DiarizationResult, confidence_threshold: float = None) -> List[AlignedSegment]` function:
   - Use ASR_CONFIDENCE_THRESHOLD if confidence_threshold is None
   - For each word in transcription segments:
     - Find speaker using find_speaker_for_word()
     - Create AlignedWord with needs_review = (confidence < threshold)
   - Group consecutive words by speaker into AlignedSegments:
     - When speaker changes, start new segment
     - Segment text = concatenation of word texts (with spaces)
     - Segment confidence = minimum confidence of contained words
     - Segment needs_review = any word needs_review
   - Return list of AlignedSegments

5. Include progress print: "Aligning X words with speaker segments...", "Alignment complete: X segments, Y flagged for review"

CRITICAL: Use temporal OVERLAP (not just overlap check) - word may span two speaker turns, assign to one with greater overlap.
CRITICAL: Handle edge case where word has no overlap with any speaker turn - assign to nearest speaker temporally.
CRITICAL: Round timestamps to 2 decimal places for cleaner output.
  </action>
