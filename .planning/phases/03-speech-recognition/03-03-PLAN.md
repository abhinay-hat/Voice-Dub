   ```python
   @dataclass
   class ASRResult:
       video_id: str
       duration: float
       detected_language: str
       language_probability: float
       num_speakers: int
       total_segments: int
       flagged_count: int
       flagged_segment_ids: List[int]
       segments: List[AlignedSegment]
       processing_time: float
       output_path: Optional[str] = None
   ```

3. Create main function `run_asr_stage(audio_path: str, video_id: str, huggingface_token: str, progress_callback: Callable[[float, str], None] = None, save_json: bool = True) -> ASRResult`:
   - Start timer for processing_time
   - Create ModelManager for sequential model loading
   - Default progress_callback to no-op if None

   - Step 1: Preprocess audio (progress 0.05)
     - preprocess_audio_for_asr(audio_path) to get 16kHz mono WAV

   - Step 2: Transcription (progress 0.10 -> 0.40)
     - transcribe_audio(preprocessed_path, model_manager)
     - Extract detected_language, language_probability from result

   - Step 3: Diarization (progress 0.45 -> 0.75)
     - diarize_audio(preprocessed_path, huggingface_token, model_manager)
     - Extract num_speakers from result

   - Step 4: Alignment (progress 0.80 -> 0.90)
     - align_transcript_with_speakers(transcription, diarization)
     - Count flagged segments (needs_review=True)

   - Step 5: Build result (progress 0.95)
     - Create ASRResult with all metadata
     - Calculate processing_time

   - Step 6: Save JSON (progress 1.0)
     - If save_json=True:
       - output_path = TEMP_DATA_DIR / f"{video_id}_transcript.json"
       - Convert ASRResult to dict (custom serialization for AlignedSegment)
       - Write JSON with ensure_ascii=False, indent=2
       - Set result.output_path

   - Step 7: Cleanup
     - model_manager.unload_current_model()
     - Delete preprocessed 16kHz WAV (cleanup temp file)

   - Return ASRResult

4. Create helper function `_asr_result_to_dict(result: ASRResult) -> dict`:
   - Convert ASRResult to JSON-serializable dict
   - Convert AlignedSegment list to list of dicts
   - Convert AlignedWord within each segment to dicts
   - Use dataclasses.asdict for simple dataclasses, custom handling for nested

## HuggingFace Token Setup (Required for Speaker Diarization)

The speech recognition pipeline uses PyAnnote for speaker diarization, which requires a HuggingFace account and API token.

### Setup Steps

1. **Create HuggingFace Account**
   - Go to https://huggingface.co/join
   - Create a free account

2. **Accept Model License**
   - Visit https://huggingface.co/pyannote/speaker-diarization-community-1
   - Click "Agree and access repository" to accept the license

3. **Generate API Token**
   - Go to https://huggingface.co/settings/tokens
   - Click "New token"
   - Name it (e.g., "voice-dub")
   - Select "Read" permission
   - Click "Generate"
   - Copy the token (starts with `hf_`)

4. **Set Environment Variable**
   ```bash
   # Windows PowerShell
   $env:HUGGINGFACE_TOKEN = "hf_your_token_here"

   # Windows CMD
   set HUGGINGFACE_TOKEN=hf_your_token_here

   # Linux/macOS
   export HUGGINGFACE_TOKEN="hf_your_token_here"
   ```

   Or create a `.env` file in the project root:
   ```
   HUGGINGFACE_TOKEN=hf_your_token_here
   ```

### Troubleshooting

- **"Repository not found" error**: You haven't accepted the model license. Visit the model page and click "Agree and access repository".
- **"Authentication required" error**: Token not set or invalid. Verify your token in HuggingFace settings.
- **Token starts with `hf_`**: This is correct. Older tokens may need to be regenerated.
```
