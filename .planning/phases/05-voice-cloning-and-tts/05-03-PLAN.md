```python
@dataclass
class AudioQualityResult:
    """Quality validation result for a single audio segment."""
    segment_id: int
    audio_path: str

    # Duration metrics
    target_duration: float
    actual_duration: float
    duration_error: float  # Percentage error
    duration_valid: bool  # Within tolerance

    # PESQ score (1.0-5.0, higher = better)
    pesq_score: Optional[float]  # None if calculation failed
    pesq_quality: str  # "excellent", "good", "fair", "poor"

    # STOI score (0.0-1.0, higher = more intelligible)
    stoi_score: Optional[float]  # None if calculation failed

    # Overall assessment
    passes_quality: bool  # True if acceptable for lip sync
    flagged_for_review: bool  # True if marginal quality
    rejection_reason: Optional[str]  # Why it failed (if applicable)
```

**QualityValidator class:**
```python
class QualityValidator:
    """Validates TTS audio quality using PESQ and STOI metrics."""

    def __init__(
        self,
        min_pesq: float = TTS_MIN_PESQ_SCORE,
        review_pesq: float = TTS_PESQ_REVIEW_THRESHOLD,
        duration_tolerance: float = TTS_DURATION_TOLERANCE
    ):
        """
        Initialize validator with quality thresholds.

        Args:
            min_pesq: Minimum PESQ score to pass (default 2.5)
            review_pesq: Flag for review if below (default 3.0)
            duration_tolerance: Acceptable duration error (default 0.05 = 5%)
        """
        self.min_pesq = min_pesq
        self.review_pesq = review_pesq
        self.duration_tolerance = duration_tolerance
```

**validate_single(
    audio_path: Path,
    reference_path: Optional[Path],
    segment_id: int,
    target_duration: float
) -> AudioQualityResult:**
- Load generated audio with librosa at 16kHz (PESQ requirement for wideband)
- Check duration against target
- If reference_path provided:
  - Calculate PESQ using pesq(16000, ref_audio, gen_audio, 'wb')
  - Calculate STOI using stoi(ref_audio, gen_audio, 16000)
- If no reference: skip PESQ/STOI (TTS doesn't have true reference for comparison)
- Determine quality tier based on scores
- Return AudioQualityResult

**validate_duration_only(
    audio_path: Path,
    segment_id: int,
    target_duration: float
) -> AudioQualityResult:**
- Simplified validation when no reference available
- Check duration accuracy only
- PESQ/STOI scores set to None
- Useful for TTS where we don't have "ground truth" to compare against

**validate_batch(
    synthesis_results: list[dict],
    reference_dir: Optional[Path] = None
) -> tuple[list[AudioQualityResult], dict]:**
- Validate all synthesized segments
- reference_dir: if provided, look for reference_{speaker_id}.wav files
- Return (results_list, summary_dict)
- Summary includes: total, passed, flagged, rejected counts

**Quality tier classification:**
- PESQ > 4.0: "excellent" (auto-accept)
- PESQ 3.0-4.0: "good" (accept)
- PESQ 2.5-3.0: "fair" (flag for review)
- PESQ < 2.5: "poor" (reject)

**Critical implementation note:**
For TTS quality validation, we typically don't have a "clean reference" to compare against (the generated audio IS the output). Two options:
1. Use reference_path as the speaker's reference sample (validates voice similarity)
2. Use duration-only validation (no PESQ/STOI)

Default to option 2 for MVP (simpler), with option 1 available for quality-critical use.

- Low RMS may indicate silence or very quiet audio
- High RMS variation may indicate distortion

- Return True if audio is mostly silent (RMS < threshold)
- Silent audio should be flagged regardless of other metrics
