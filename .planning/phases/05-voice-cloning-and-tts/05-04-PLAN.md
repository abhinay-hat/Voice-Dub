   validator = QualityValidator()
   quality_results, quality_summary = validator.validate_batch(
       synthesis_results,
       reference_dir=output_dir  # Use speaker references for emotion comparison
   )

   # Process validation results - update segment metadata
   rejected_count = 0
   for segment, quality in zip(segments, quality_results):
       segment.quality_passed = quality.passes_quality
       segment.flagged_for_review = quality.flagged_for_review
       segment.rejection_reason = quality.rejection_reason

       )
       segment.pitch_variance_ratio = quality.pitch_variance_ratio

       if not quality.passes_quality:
           rejected_count += 1

   # Fail stage if too many segments rejected
   if rejected_count > len(segments) * 0.5:
       raise TTSStageFailed(
           f"Quality validation failed: {rejected_count}/{len(segments)} segments rejected. "
           f"Check audio quality and reference samples."
       )

   # Log quality summary
   logger.info(f"Quality validation: {quality_summary['passed']}/{quality_summary['total']} passed, "
               f"{quality_summary['flagged']} flagged, {quality_summary['rejected']} rejected")

**## Phase 5: Voice Cloning & TTS**

```markdown
### Overview

Phase 5 generates English audio from translated text using XTTS-v2 voice cloning. Each speaker's voice is cloned from a 6-10 second reference sample extracted from the original video, preserving voice characteristics and emotional tone.

### Components

- **Reference Extractor**: Selects cleanest audio samples per speaker
- **Speaker Embeddings**: XTTS conditioning latents for voice cloning
- **XTTS Generator**: Synthesis with duration matching
- **Quality Validator**: PESQ-based quality assessment with emotion preservation check

### Emotion Preservation

XTTS-v2 preserves emotional characteristics through speaker conditioning latents extracted from reference audio. The quality validator checks emotion preservation by comparing pitch variance between generated and reference audio:

- **Pitch variance ratio 0.8-1.2**: Emotion well preserved
- **Pitch variance ratio 0.6-0.8 or 1.2-1.5**: Marginal - flagged for review
- **Pitch variance ratio <0.6 or >1.5**: Emotion lost/exaggerated - flagged for review

### Usage

```python
from src.stages.tts_stage import run_tts_stage

result = run_tts_stage(
    translation_json_path="data/temp/video_translation.json",
    audio_path="data/temp/video_audio.wav",
    output_dir="data/temp/tts_output",
    progress_callback=lambda p, s: print(f"[{p*100:.0f}%] {s}")
)

print(f"Synthesized {result.successful_segments}/{result.total_segments} segments")
print(f"Flagged for review: {result.flagged_count}")
print(f"Emotion issues: {result.emotion_flagged_count}")
```

### Configuration

TTS settings in `src/config/settings.py`:
- `TTS_TEMPERATURE`: Creativity control (default 0.65)
- `TTS_DURATION_TOLERANCE`: Timing accuracy (default 5%)
- `TTS_MIN_PESQ_SCORE`: Quality threshold (default 2.5)

### Requirements

XTTS-v2 requires ~2GB VRAM for inference. Reference samples need 6-10 seconds of clean audio per speaker.

**Non-Commercial License**: XTTS-v2 is licensed under CPML for non-commercial use only.
```
