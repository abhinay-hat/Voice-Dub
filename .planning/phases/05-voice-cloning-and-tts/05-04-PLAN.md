```python
@dataclass
class TTSResult:
    """Complete TTS stage output."""
    video_id: str
    total_segments: int
    successful_segments: int
    failed_segments: int
    flagged_count: int
    flagged_segment_ids: List[int]
    avg_duration_error: float
    segments: List[SynthesizedSegment]
    processing_time: float
    output_dir: Optional[str] = None

    def to_dict(self):
        return asdict(self)
```

**run_tts_stage(
    translation_json_path: str,
    audio_path: str,
    output_dir: Optional[str] = None,
    progress_callback: Optional[Callable[[float, str], None]] = None
) -> TTSResult:**

Pipeline steps (with progress percentages):
1. (0.05) Load translation JSON
2. (0.10) Create output directory if needed
3. (0.15) Extract reference samples per speaker
4. (0.25) Generate speaker embeddings
5. (0.35-0.80) Synthesize all segments with duration matching
   - Group by speaker for efficiency
   - Report per-segment progress
6. (0.85) Validate audio quality
7. (0.90) Build result structure
8. (0.95) Export result JSON (segment manifest)
9. (1.0) Cleanup (unload model, clear CUDA cache)

**try-finally cleanup pattern** (matches translation_stage.py):
- Model unload in finally block
- CUDA cache clear
- Log cleanup completion

**Error handling:**
- If reference extraction fails for speaker: log warning, use generic voice or skip
- If synthesis fails for segment: log error, mark as failed, continue
- If >50% segments fail: raise TTSStageFailed exception

**Output directory structure:**
```
output_dir/
  reference_SPEAKER_00.wav  # Reference samples
  reference_SPEAKER_01.wav
  segment_0.wav             # Synthesized segments
  segment_1.wav
  ...
  tts_result.json           # Result manifest
```

**## Phase 5: Voice Cloning & TTS**

```markdown
### Overview

Phase 5 generates English audio from translated text using XTTS-v2 voice cloning. Each speaker's voice is cloned from a 6-10 second reference sample extracted from the original video, preserving voice characteristics and emotional tone.

### Components

- **Reference Extractor**: Selects cleanest audio samples per speaker
- **Speaker Embeddings**: XTTS conditioning latents for voice cloning
- **XTTS Generator**: Synthesis with duration matching
- **Quality Validator**: PESQ-based quality assessment

### Usage

```python
from src.stages.tts_stage import run_tts_stage

result = run_tts_stage(
    translation_json_path="data/temp/video_translation.json",
    audio_path="data/temp/video_audio.wav",
    output_dir="data/temp/tts_output",
    progress_callback=lambda p, s: print(f"[{p*100:.0f}%] {s}")
)

print(f"Synthesized {result.successful_segments}/{result.total_segments} segments")
print(f"Flagged for review: {result.flagged_count}")
```

### Configuration

TTS settings in `src/config/settings.py`:
- `TTS_TEMPERATURE`: Creativity control (default 0.65)
- `TTS_DURATION_TOLERANCE`: Timing accuracy (default 5%)
- `TTS_MIN_PESQ_SCORE`: Quality threshold (default 2.5)

### Requirements

XTTS-v2 requires ~2GB VRAM for inference. Reference samples need 6-10 seconds of clean audio per speaker.

**Non-Commercial License**: XTTS-v2 is licensed under CPML for non-commercial use only.
```
