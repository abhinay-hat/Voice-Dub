<objective>
Replace the sequential Whisper transcription with faster-whisper's BatchedInferencePipeline for parallel segment decoding, and add StageTimer instrumentation to measure ASR performance.

Purpose: BatchedInferencePipeline processes multiple audio segments in parallel on the GPU, yielding 2-4x speedup on long audio. For a 20-minute video, this can reduce ASR time from ~3 minutes to ~1 minute. Adding StageTimer instrumentation enables measuring the actual GPU time for validation against the 10-20 minute total pipeline target.

Output: Updated transcription.py using BatchedInferencePipeline, instrumented asr_stage.py, and unit tests.
</objective>
