**Add new method: translate_with_candidates(segment_text: str, source_lang: str, num_candidates: int = 3) -> tuple[list[str], list[float]]**

Implementation:
1. Ensure model is loaded via _ensure_model_loaded()
2. Encode input with processor:
   ```python
   inputs = self.processor(
       text=segment_text,
       src_lang=source_lang,
       return_tensors="pt"
   ).to('cuda')
   ```
3. Generate with beam search:
   ```python
   outputs = self.model.generate(
       **inputs,
       tgt_lang="eng",
       num_beams=num_candidates,           # Beam width
       num_return_sequences=num_candidates, # Return all beams
       return_dict_in_generate=True,        # Get metadata
       output_scores=True,                  # Get confidence scores
       max_new_tokens=512
   )
   ```
4. Extract confidence scores using compute_transition_scores():
   ```python
   import torch
   transition_scores = self.model.compute_transition_scores(
       outputs.sequences,
       outputs.scores,
       normalize_logits=True
   )

   confidences = []
   for i in range(len(outputs.sequences)):
       # Average log probability across tokens, convert to probability
       avg_log_prob = transition_scores[i].mean().item()
       confidence = torch.exp(torch.tensor(avg_log_prob)).item()
       confidences.append(confidence)
   ```
5. Decode all candidates:
   ```python
   candidates = [
       self.processor.decode(seq, skip_special_tokens=True)
       for seq in outputs.sequences
   ]
   ```
6. Return tuple: `(candidates, confidences)` - lists of same length

**Add new method: translate_batch(segments: list[str], source_lang: str, num_candidates: int = 1) -> list[dict]**

Implementation:
1. Ensure model is loaded
2. Batch encode (up to 8 segments at a time for memory efficiency):
   ```python
   import torch
   batch_size = 8
   all_results = []

   for i in range(0, len(segments), batch_size):
       batch = segments[i:i+batch_size]
       inputs = self.processor(
           text=batch,
           src_lang=source_lang,
           return_tensors="pt",
           padding=True
       ).to('cuda')

       # Generate for batch
       outputs = self.model.generate(
           **inputs,
           tgt_lang="eng",
           num_beams=num_candidates,
           num_return_sequences=num_candidates,
           return_dict_in_generate=True,
           output_scores=True,
           max_new_tokens=512
       )

       # Compute confidence scores for all outputs
       transition_scores = self.model.compute_transition_scores(
           outputs.sequences,
           outputs.scores,
           normalize_logits=True
       )

       # Extract candidates and scores per input
       # (outputs has batch_size * num_candidates sequences)
       for j in range(len(batch)):
           candidates = []
           scores = []
           for k in range(num_candidates):
               idx = j * num_candidates + k
               translation = self.processor.decode(
                   outputs.sequences[idx],
                   skip_special_tokens=True
               )
               candidates.append(translation)

               # Extract confidence score for this candidate
               avg_log_prob = transition_scores[idx].mean().item()
               confidence = torch.exp(torch.tensor(avg_log_prob)).item()
               scores.append(confidence)

           all_results.append({
               "segment_text": batch[j],
               "candidates": candidates,
               "scores": scores
           })
   ```
3. Return list of dicts (one per input segment)

```python
TRANSLATION_APPROX_CHARS_PER_TOKEN = 4  # Rough estimate for token counting
```

**ContextChunker class:**

Structure:
- `__init__(self, max_tokens=None, overlap_tokens=None)`
  - Default from settings constants
  - Store chunking parameters

Methods:

1. **chunk_transcript_with_overlap(segments: list[dict]) -> list[dict]**
   - Input: List of transcript segments with "text" and "duration" keys
   - Process:
     1. Estimate tokens for each segment: `len(segment["text"]) // TRANSLATION_APPROX_CHARS_PER_TOKEN`
     2. Group segments into chunks maintaining token limit
     3. When chunk exceeds max_tokens, save current chunk and start new with overlap
     4. Overlap: include last N segments from previous chunk (where total < overlap_tokens)
   - Return: List of chunks, each chunk is:
     ```python
     {
         "segments": list[dict],  # Segment dicts with text, duration, metadata
         "start_idx": int,        # Index of first segment in original list
         "end_idx": int,          # Index of last segment in original list
         "total_tokens": int,     # Approximate token count
         "has_overlap": bool      # True if this chunk includes overlap from previous
     }
     ```

2. **merge_translated_chunks(chunks: list[dict]) -> list[dict]**
   - After translation, merge overlapping chunks back to original segment list
   - For overlapped segments (appear in multiple chunks), take translation from later chunk (has more context)
   - Return: List of translated segments in original order, duplicates removed

**Helper functions (module-level):**

**chunk_transcript_with_overlap(segments: list[dict], max_tokens: int = None, overlap_tokens: int = None) -> list[dict]**
- Convenience function that creates ContextChunker and calls method
- Allows one-line usage

**merge_translated_chunks(translated_segments: list[dict], chunks: list[dict]) -> list[dict]**
- Convenience function that creates ContextChunker and calls method
- Matches the pattern from chunk_transcript_with_overlap()
- Allows one-line usage without instantiating ContextChunker

**Implementation details:**
- Token estimation is approximate (chars/4), prefer conservative chunking
- Track segment indices to enable merging after translation
- If transcript fits in single chunk, return as single-chunk list (no chunking needed)
- Validate inputs: segments must have "text" key
- Handle edge cases: empty segments list (return empty), single segment (return single chunk)
