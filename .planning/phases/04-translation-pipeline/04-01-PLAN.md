**Class structure:**
- `__init__(self, model_manager: ModelManager)` - Store manager reference, defer model loading
- `_ensure_model_loaded()` - Lazy load SeamlessM4T + processor via ModelManager
- `translate_segment(segment_text: str, source_lang: str) -> dict` - Single segment translation

**Key implementation requirements:**

1. **Lazy model loading** (don't load in __init__, load on first translate call):
```python
if self.model is None:
    self.processor = AutoProcessor.from_pretrained(SEAMLESS_MODEL_ID)
    self.model = self.model_manager.load_model(
        "seamless_m4t_translation",
        lambda: SeamlessM4Tv2ForTextToText.from_pretrained(
            SEAMLESS_MODEL_ID,
            torch_dtype=torch.float16  # Half precision for VRAM efficiency
        ).to('cuda')
    )
```

2. **Single segment translation** (greedy decoding for speed):
- Accept segment text and source language code (use SeamlessM4T 3-letter codes: "jpn", "kor", "cmn", etc.)
- Encode with processor: `processor(text=segment_text, src_lang=source_lang, return_tensors="pt").to('cuda')`
- Generate with greedy: `model.generate(**inputs, tgt_lang="eng", max_new_tokens=512)` (no beam search yet - Plan 2 adds that)
- Decode: `processor.decode(output[0], skip_special_tokens=True)`
- Return dict with keys: `translation`, `source_lang`, `target_lang`

- `from src.models.model_manager import ModelManager`

- `from transformers import AutoProcessor, SeamlessM4Tv2ForTextToText`
- `import torch`

4. **Error handling:**
- Catch model loading errors with helpful messages
