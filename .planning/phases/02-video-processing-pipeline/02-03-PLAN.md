**process_video_ui(video_path: str, progress=gr.Progress()) -> str:**
- Main handler for Gradio interface
- Validates input video using validate_video_file()
- Shows video info (resolution, duration, format) before processing
- Wraps process_video() with progress callback that maps to gr.Progress()
- Returns output video path for Gradio to display
- Handles errors gracefully (returns None with gr.Warning for user feedback)

**Progress integration:**
```python
def progress_callback(pct: float, desc: str):
    progress(pct, desc=desc)
```

**Interface definition:**
```python
with gr.Blocks(title="Voice Dub - Video Processing") as demo:
    gr.Markdown("# Voice Dub - Video Processing")
    gr.Markdown("Upload a video to test the processing pipeline. Currently performs round-trip (extract audio/video, merge back) to validate FFmpeg toolchain.")

    with gr.Row():
        with gr.Column():
            input_video = gr.Video(
                sources=["upload"],
                label="Upload Video",
                format="mp4"  # Browser playback format
            )
            process_btn = gr.Button("Process Video", variant="primary")
            video_info = gr.Textbox(label="Video Info", interactive=False)

        with gr.Column():
            output_video = gr.Video(label="Processed Video")
            status = gr.Textbox(label="Status", interactive=False)

    # Event handlers
    input_video.change(fn=display_video_info, inputs=input_video, outputs=video_info)
    process_btn.click(fn=process_video_ui, inputs=input_video, outputs=[output_video, status])
```

**display_video_info(video_path: str) -> str:**
- Called when video is uploaded
- Uses get_video_info() to show: resolution, duration, codec, format
- Format: "1920x1080 | H.264 | 00:05:30 | MP4"
