```python
@dataclass
class AssemblyResult:
    """Result of audio-video assembly stage."""
    output_path: Path
    total_duration: float
    segment_count: int
    sync_checkpoints: List[SyncCheckpoint]
    drift_detected: bool
    max_drift_ms: float
    sample_rate_normalized: bool
    original_sample_rates: List[int]  # Sample rates before normalization
    processing_time: float
    video_fps: float

    def to_dict(self) -> dict:
        """Serialize for JSON export."""
        return {
            'output_path': str(self.output_path),
            'total_duration': self.total_duration,
            'segment_count': self.segment_count,
            'checkpoints': [
                {
                    'timestamp': cp.timestamp,
                    'drift_ms': cp.drift_ms,
                    'frame_number': cp.frame_number,
                    'within_tolerance': cp.within_tolerance
                }
                for cp in self.sync_checkpoints
            ],
            'drift_detected': self.drift_detected,
            'max_drift_ms': self.max_drift_ms,
            'sample_rate_normalized': self.sample_rate_normalized,
            'processing_time': self.processing_time,
            'video_fps': self.video_fps
        }
```

Pipeline flow with progress points:
1. (0.05) Load TTS result JSON, extract segment metadata
2. (0.10) Create TimedSegment objects from TTS output
3. (0.15) Validate timestamp precision (validate_timestamps_precision)
4. (0.25) Normalize all audio segments to 48kHz (batch_normalize)
   - Track original sample rates for reporting
5. (0.45) Concatenate normalized segments (concatenate_audio_segments)
   - Handle gaps between segments with silence padding
6. (0.60) Validate sync at 5-minute intervals (validate_sync_at_intervals)
   - If drift > 45ms at any checkpoint: log warning (don't fail)
7. (0.80) Merge video with audio (merge_with_sync_validation)
   - Use async_correction=True for clock drift protection
8. (0.95) Export assembly_result.json to output directory
9. (1.0) Cleanup temp files (normalized audio, concatenated audio)

**Error handling:**
- If TTS result JSON missing/invalid: raise AssemblyStageFailed
- If no segments in TTS result: raise AssemblyStageFailed
- If sample rate normalization fails: raise AssemblyStageFailed
- If FFmpeg merge fails: raise AssemblyStageFailed
- If drift > 45ms: log warning, set drift_detected=True, continue (user decides)

### Phase 6: Audio-Video Assembly

Frame-perfect audio-video synchronization for dubbed videos.

**Components:**
- **Timestamp Validator**: Float64 precision validation prevents drift accumulation
- **Audio Normalizer**: Sample rate normalization to 48kHz (video production standard)
- **Audio Concatenator**: Segment concatenation with gap handling
- **Drift Detector**: Sync validation at 5-minute intervals (ATSC 45ms tolerance)
- **Video Merger**: FFmpeg merge with `-af aresample=async=1` for clock drift correction

**Usage:**
```python
from src.stages.assembly_stage import run_assembly_stage

result = run_assembly_stage(
    video_path=Path("input_video.mp4"),
    tts_result_path=Path("tts_result.json"),
    output_path=Path("dubbed_video.mp4"),
    video_fps=30.0,
    progress_callback=lambda p, m: print(f"{p*100:.0f}%: {m}")
)

print(f"Output: {result.output_path}")
print(f"Drift detected: {result.drift_detected}")
print(f"Max drift: {result.max_drift_ms:.2f}ms")
for cp in result.sync_checkpoints:
    print(f"  {cp.timestamp/60:.0f}min: {cp.drift_ms:+.2f}ms")
```

**Pipeline Flow:**
1. (0.05) Load TTS result JSON
2. (0.10-0.15) Validate timestamp precision
3. (0.25) Normalize audio to 48kHz
4. (0.45) Concatenate segments
5. (0.60) Validate sync at 5-minute intervals
6. (0.80) Merge with FFmpeg sync flags
7. (0.95) Export result JSON
8. (1.0) Cleanup

**Sync Validation:**
- Checkpoints at 5-minute intervals (300s)
